Perguntas:¶

O que significa enrgia fotovoltaica?

  • Energia fotovoltaica a energia obtida a partir da converso direta da luz solar, atravz das clulas fotovoltaicas. A clula fotovoltaica um dispositivo fabricado de materiais semicondutores que possibilita o aproveitamento do efeito fotoeltrico.

  • O efeito fotoeltrico foi explicado por Albert Einstein (o que deu a ele o Premio Nobel de Fisica). Esse efeito consiste na emisso de eltrons por um material, quando este exposto a uma radiao eletromagntica (luz, por exemplo).

  • A celula fotovoltaica apresenta a estrutura semelhante a de um diodo, com duas camadas (P e N) sobrepostas. A luz incide sobre a camada N, que emite eletrns para a camada P, gerando uma diferena de potncial.

O que so sries temporais?

  • Series temporais so sequncias de numeros indexados por sua ordem cronolgica. Alguns exemplos de series temporais so: Incidencia mensal de chuvas, preos de criptomoedas, registros de eletroencefalograma.

  • As series temporais so importantes em diversas areas do conhecimento e a partir disso desenvolveu-se o campo de anlise de series temporais, que desenvolve mtodos para obteno de informao util a partir das series temporais.

  • Para gerar predies da serie temporal em questo poderiamos utilizar um mtodo tradicional, como o ARIMA (ou Seasonal ARIMA adaptado para series com componentes de sasonalidade) ou ento tecnicas que utilizam redes CNN ou LSTM. As CNNs para series temporais realizam a convoluo do sinal com filtros que so aprendidos ao longo do treinamento, e as LSTMs, que foram projetadas para series temporais, contm componentes de memria em suas unidades, fazendo-as adequadas para reconhecer padres temporais complexos.

Como a SolarView utiliza dados para gerar valor para seus clientes?

  • A SolarView coleta dados de unidades de gerao de energia solar, utilizando dataloggers e medidores (com hardware prprio que pode ser integrado a uma ampla gama de inversores de frequncia), e os disponibiliza de forma agregada em plataformas Web/Mobile.

  • Dentre as principais funcionalidades oferecidas pelo produto esto a gesto centralizada de todo sistema de gerao de energia, independente do fabricante dos equipamentos, a auditoria de valores de leitura e consumo antes mesmo da chegada da fatura mensal e gesto inteligente da manuteno.

  • Outras entregas interessantes poderiam ser realizadas cruzando informaes de predies por demanda de energia, predies de quantidade de energia produzida com os indices de produao dos clientes e tambm no auxilio a escolha de local para futuras instalaes de gerao de eergia solar.

CREATE TABLE `solar_view`.`allsky_insolation_hs` ( 
    `id` INT NOT NULL AUTO_INCREMENT , 
    `date` DATE NULL DEFAULT NULL , 
    `latitude` FLOAT NULL DEFAULT NULL , 
    `longitude` FLOAT NULL DEFAULT NULL , 
    `insolation` FLOAT NULL DEFAULT NULL , 
    PRIMARY KEY (`id`),
    UNIQUE `measurement` (`latitude`, `date`, `longitude`)) 
ENGINE = InnoDB;

Observações¶

A modelagem utiliza uma chave primaria chamada id com incremento automtico, e uma chave composta unica chamada meassurement que assegura que assegura que s haja uma entrada de insolation no banco para cada uma das combinaes dos campos latitude, date, longitude.

Criação da query e da conexão com o banco¶

A funo insert_data abre e fecha uma conexo com o banco para cada entrada a ser inserida. Foi feita dessa forma pois a conexo com o banco no thread-safe, o que gera problemas quando o objeto utilizado por mais de um processo.

In [2]:
add_entry = ('INSERT INTO `allsky_insolation_hs` '
             '(`date`, `latitude`, `longitude`, `insolation`)'
             ' VALUES (%(date)s, %(LAT)s, %(LON)s, %(ALLSKY_SFC_SW_DWN)s)')


def connect():
    return connection.MySQLConnection(
        user='root',
        password='admin',
        database='solar_view'
    )


def insert_data(data):
    cnx = connect()
    cursor = cnx.cursor()
    data = {k: float(v) for k, v in dict(data[1]).items()}
    try:
        cursor.execute(add_entry, data)
    except mysql.connector.IntegrityError as err:
        pass
    cnx.commit()
    cursor.close()
    cnx.close()
